## Overview

The **Ordering and Billing System** is a Windows desktop application developed to simplify restaurant operations by managing orders, billing, inventory, staff, and sales reports through an intuitive, easy-to-navigate interface.

The application emphasizes **usability**, **performance**, and **efficient business workflow management**, making it well suited for restaurants, cafés, and small food businesses looking to digitize their day-to-day operations.

<br>

## Features

| Category | Capabilities |
|---|---|
| **Authentication & Access** | User authentication, role-based access control (Admin & Staff) |
| **Sales & Orders** | Point of Sale (POS), order management, table management |
| **Billing** | Billing & payment processing, printable receipts, transaction history |
| **Inventory** | Product management, category management |
| **Operations** | Staff management, system settings, database backup & restore |
| **Insights** | Sales reports, real-time dashboard |

## Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/tristanbon/ordering-billing-system.git
   ```
2. **Open the solution**
   Open `OrderingAndBillingSystem.sln` in Visual Studio 2019.
3. **Restore dependencies**
   Restore NuGet packages if prompted.
4. **Set up the database**
   Import the provided SQL database from the `Database/` folder.
5. **Configure the connection**
   Update the database connection string in `App.config`.
6. **Build and run**
   Build the solution and launch the application.
